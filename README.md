## Project Overview

### Description
Для работы был использован открытый датасет с набором транзакционных данных  британской онлайн-розничной компании, за период с 01.12.2010 по 09.12.2011: https://archive.ics.uci.edu/dataset/352/online%2Bretail
Компания специализируется на продаже подарочной продукции. Многие клиенты компании являются оптовиками.

### Проделанная работа:
*    Постановка целей проекта и разработка плана его реализации;
*    Разработка гипотез;
*    Подготовка данных: обработка пропусков, выбросов, некорректных значения;
*    Анализ динамических показателей: revenue, arpu, mau, количество уникальных пользователей, количество уникальных заказов;
*    Когортный анализ;
*    Анализ истории покупок и доли выручки, приносимой клиентами разных категорий;
*    RFM-анализ;
*    Кластеризация на основе KMeans;
*    ABC-анализ;
*    Анализ отмененных заказов;
*    Проверка гипотез, сформулированных на основе выполненного анализа.
